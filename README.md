## 📌 Sobre o Projeto
Este repositório foi desenvolvido como parte prática do **Trabalho de Introdução à Ciência de Dados**. O objetivo do projeto é estruturar um ecossistema inteligente voltado à análise de dados financeiros através do desenvolvimento de scripts em Python (`projeto.py` e `bot_finance2.py`).

O projeto combina automação, coleta e manipulação de indicadores econômicos com foco em fornecer insights claros sobre o cenário financeiro atual.
